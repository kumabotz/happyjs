# A happy micro-framework
For the happy souls.

## Setup
`npm install`

## Run
`npm start`

## Roll it yourself

1. `npm init`
1. `npm i -D browserify http-server testem mocha babel babelify expect`
1. Update files:
  - package.json
1. Add files:
  - index.html
  - index.js
  - add.js
  - add.spec.js
  - testem.yml
1. `npm start`
